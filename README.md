# Вращение геометрических фигур относительно произвольной точки

Программа представляет собой диалоговое приложение MFC с GDI+, которое позволяет отображать геометрические фигуры (прямоугольник и окружность) с возможностью их поворота вокруг задаваемой пользователем точки. Приложение демонстрирует два различных подхода к реализации одного и того же функционала, разработанных в параллельных ветках. Программа существует в двух вариантах, которые визуально и функционально идентичны, но реализуют механизм взаимодействия с пользователем по-разному.

## Различия в реализациях

### Ветка Rotate (класс Rotate):
- Обработка клика мыши инкапсулирована внутри класса-рисовальщика `Rotate`
- Прямой доступ к родительскому диалогу через `GetParent()`
- Компактная логика: элемент сам управляет своей реакцией на нажатие

### Ветка Dialog (класс Drawer):
- Обработка клика вынесена в диалоговое окно (`CMFCApplication6Dlg::OnLButtonDown`)
- Дополнительная проверка попадания в область рисования через `PtInRect`
- Более явная и развернутая структура обработки событий

## Функциональные возможности

1. **Отображение фигур:**
   - Прямоугольник с линейным градиентом (голубой → пурпурный)
   - Окружность с линейным градиентом (белый → оранжевый)
   - Контуры фигур выделены цветами (зеленый для круга, синий для прямоугольника)

2. **Управление поворотом:**
   - Слайдер с диапазоном от -45° до +45°
   - Текстовое поле для отображения текущего угла
   - Плавный поворот всех фигур относительно точки вращения

3. **Выбор точки вращения:**
   - Клик левой кнопкой мыши по области рисования
   - Отображение координат точки в полях X и Y
   - Автоматический сброс угла поворота при смене точки
